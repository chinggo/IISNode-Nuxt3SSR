# Note Nuxt3 Publish to IISNode Process

## Nuxt3 Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Check the IISNode dependencies was install
1. Nodejs
2. IISNode Module
3. URL Rewrite
